# Aplicação de Machine Learning na Detecção de Fraudes Bancárias

Projeto desenvolvido como Trabalho de Conclusão de Curso em Ciência da Computação.

## Objetivo

Aplicar algoritmos de Machine Learning na detecção de fraudes bancárias utilizando um dataset público e desbalanceado.

## Algoritmos Utilizados

- Decision Tree
- SVM
- MLPClassifier
- XGBoost

## Tecnologias

- Python 3
- Pandas
- NumPy
- Scikit-Learn
- XGBoost

## Como executar

### 1. Criar ambiente virtual

```bash
python -m venv venv
```

### 2. Ativar ambiente virtual

Windows:

```bash
venv\Scripts\activate
```

Linux/Mac:

```bash
source venv/bin/activate
```

### 3. Instalar dependências

```bash
pip install -r requirements.txt
```

### 4. Executar

```bash
python src/main.py
```

## Dataset

https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud
